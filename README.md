# SFARNNI - Flight & Hotel Booking Website

A full-stack booking platform that allows users to search, book, and manage flights and hotels through a single unified system.

## Features

### User Features

* User Registration & Login
* Secure Authentication & Authorization
* Search Flights
* Search Hotels
* Online Booking System
* Stripe Payment Integration
* Booking Confirmation

### Admin Features

* Manage Flights
* Manage Hotels
* Manage Users
* Block/Delete Accounts
* Dashboard Analytics

## Technologies Used

### Backend

* ASP.NET Core
* Entity Framework Core
* ASP.NET Identity
* Repository Pattern
* Unit of Work Pattern
* SQL Server

### Frontend

* HTML
* CSS
* JavaScript
* Bootstrap

### Payment Gateway

* Stripe API

## Architecture

The application follows Clean Architecture principles and implements:

* Repository Pattern
* Unit of Work Pattern
* ASP.NET Core Areas
* Role-Based Authorization
